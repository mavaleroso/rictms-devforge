# Blank Laravel + Inertia + React

This is your academy sandbox. Navigate the files on the left, update the ones
highlighted by the task, then Run / Submit to check assertions.

## Layout

- `routes/web.php` — named routes
- `app/Http/Controllers/` — thin controllers (Inertia responses only)
- `resources/js/pages/` — Inertia React pages
- `resources/js/layouts/` — shared layouts
- `resources/js/components/` — reusable UI

## Rules

1. Keep controllers thin — no business logic.
2. Prefer typed props and clear component names.
3. Do not invent new folders unless the task asks for them.
